# 23 - Speech Synthesis

- 입력된 텍스트를 다양한 음성, 속도, 톤으로 읽어주는 웹사이트

## 찾아본 내용

```
1. speechSynthesis
    > 음성 서비스를 위한 컨트롤러 인터페이스

1. SpeechSynthesisUtterance
    > speechSynthesis를 제어(음성, 속도, 톤 등)하기 위한 다양한 옵션이 포함되어 있다.
```
